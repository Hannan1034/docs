### Body Parameters

<ParamField body="apiKey" type="string" required>
  Your API key for authentication
</ParamField>

<ParamField body="candidateId" type="string" required>
  The unique ID of the candidate to send the interview link
</ParamField>

### Functionality
This API sends an interview invitation email to the candidate with a unique interview link. 

If the candidate doesn't take the interview within 3 days, an automatic follow-up will be sent via email and SMS to remind them.

<RequestExample>
```json Request
{
  "apiKey": "your_api_key",
  "candidateId": "987fcdeb-51a2-12d3-a456-426614174000"
}
```
</RequestExample>

<ResponseExample>
```json Response
{
  "status": "Success",
  "message": "Interview sent successfully",
  "interviewLink": "https://www.internguys.com/interview/987fcdeb-51a2-12d3-a456-426614174000"
}
```
</ResponseExample>
