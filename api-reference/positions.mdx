### Body Parameters

<ParamField body="apiKey" type="string" required>
  Your API key for authentication
</ParamField>

<ParamField body="position" type="string" required>
  Title of the position
</ParamField>

<ParamField body="location" type="string" required>
  Physical location of the position (can also be "Remote")
</ParamField>

<ParamField body="type" type="string" required>
  Type of position ("Full-Time", "Part-Time", "Intern", "Contractor")
</ParamField>

<ParamField body="jobDescription" type="string" required>
  Detailed description of the position
</ParamField>

<ParamField body="customInterviewScript" type="object" required>
  Customize the interview script to control Noraâ€™s questions for candidates
  <Expandable title="How does it work?">
    `Buckets` define dynamic question categories that adapt to candidate responses. 
    
    Choose from:  

    *Candidate's Relevant Experiences, In-depth Experience Questions, Role-specific Scenario Questions, Role-specific Technical Questions, and Cultural Fit Questions.*  

    `Questions` are fixed questions that Nora will ask exactly as provided.
  </Expandable>
</ParamField>

<ParamField body="autoInterview" type="boolean">
  Whether to automatically send Nora interviews to candidates
</ParamField>

<RequestExample>
```json Request
{
  "apiKey": "your_api_key",
  "position": "Software Engineer",
  "location": "San Francisco, CA",
  "type": "Full-Time",
  "jobDescription": "We are looking for a Software Engineer...",
  "customInterviewScript": {
    "Buckets": [
      { "Name": "Candidate's Relevant Experiences" },
      { "Name": "In-depth Experience Questions" },
      { "Name": "Role-specific Technical Questions" }
    ],
    "Questions": [
      "Are you open to working on-site at our San Francisco office?",
      "Does a salary range of $120K to $200K align with your expectations?",
      "What makes you a good fit for this position?"
    ]
  },
  "autoInterview": false
}
```
</RequestExample>

```json Response
{
    "status": "Success",

    "positionId": "123e4567-e89b-12d3-a456-426614174000",
}
```
</ResponseExample>
