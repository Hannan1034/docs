---

## Getting Your API Key

1. Log in to your [Nora AI Dashboard](https://recruiter.internguys.com/dashboard)
2. Click "Create API Key"

<Warning>
Keep your API key secure and never share it publicly.
</Warning>

## Using Your API Key

Include your API key in all API requests. Example:

```json
{
  "apiKey": "your_api_key",
  "otherParameters": "values"
}
