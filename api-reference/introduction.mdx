# Welcome to Nora AI

Nora AI provides APIs for automating your interview process. Our APIs allow you to:

- Create positions with custom interview scripts
- Process candidate applications
- Send automated AI interviews
- Review interview scores and results

## Getting Started

1. Get your API key from the Nora AI dashboard
2. Make your first API call to create a position
3. Start processing candidates
